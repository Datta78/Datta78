## ◈ Projects

<br/>

#### 💸 Expense Manager
Full-stack finance tracker with budget goals, category-wise spending, Chart.js visual reports, and an alert system that fires when limits are crossed. Session-based auth with basic security hardening.
`PHP` · `MySQL` · `Chart.js` · `Bootstrap`

<br/>

#### 🤖 WhatsApp Auto-Reply Bot
Android automation bot built with Java and the Accessibility API — no third-party API or WhatsApp Business approval needed. Keyword-triggered replies, configurable time windows, silent background execution. Built for businesses that need responsiveness without extra headcount.
`Java` · `Android SDK` · `Accessibility API`

<br/>

#### 🛒 Online Food Ordering System
Dual-panel platform for restaurants and canteens. Customers browse, cart, and checkout. Admins manage the menu, track orders, and view revenue. Full order pipeline with live status updates across both panels.
`PHP` · `MySQL` · `JavaScript` · `Bootstrap`

<br/>

#### 📋 Student Attendance System
Role-based attendance platform for institutions — Admin, Teacher, and Student panels with distinct permissions. Auto-calculates attendance percentages per subject, flags students below threshold, and exports records for reporting.
`PHP` · `MySQL` · `Bootstrap` · `JavaScript`
