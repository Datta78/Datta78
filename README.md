## 🚀 Projects

<br/>

### 💸 Expense Manager Web App
> *Personal finance management platform with intelligent budgeting and visual analytics*

A full-featured finance tracker built to handle real-world budgeting workflows. Supports multiple expense categories, tracks against monthly budget goals, and fires alerts when thresholds are crossed. Reports visualised using Chart.js with export options. Authentication handled via PHP sessions with security hardening.

| Feature | Details |
|:---|:---|
| 📊 Visual Reports | Bar, pie, and line charts for spending breakdown via Chart.js |
| 🔔 Alert System | Triggers notifications when budget limits are exceeded |
| 📁 Data Export | Downloadable reports in CSV format |
| 🔐 Auth | Session-based login with input validation and SQL injection prevention |
| 🧩 Stack | `PHP` `MySQL` `Chart.js` `Bootstrap` `JavaScript` |

<br/>

### 🤖 WhatsApp Auto-Reply Bot
> *Android-based automation engine for WhatsApp message handling*

Built natively for Android using Java and the Accessibility API — no third-party WhatsApp Business API required. Monitors incoming messages in the background and triggers automated replies based on keyword rules and scheduling configurations. Built for businesses needing lightweight automation without API costs or approval overhead.

| Feature | Details |
|:---|:---|
| ⚡ Keyword Engine | Rule-based keyword matching for contextual auto-replies |
| 🕐 Scheduling | Time-window configuration — replies only during set hours |
| 📋 Template System | Customisable message templates per contact or group type |
| 🔕 Background Mode | Silent execution using Android Accessibility Service |
| 🧩 Stack | `Java` `Android SDK` `Accessibility API` `SharedPreferences` |

<br/>

### 🛒 Online Food Ordering System
> *Full-stack web platform with dual-role panel architecture*

A complete food ordering solution for restaurants and canteens. Split into a customer-facing ordering interface and a back-end admin panel for menu and order management. Orders flow through real-time status updates visible to both sides. Schema supports multiple branches with independent menu configuration.

| Feature | Details |
|:---|:---|
| 👤 Customer Panel | Browse menu, add to cart, place orders, track status |
| 🔧 Admin Panel | Manage menu items, view & update order status, revenue summary |
| 🔄 Order Pipeline | Placed → Preparing → Ready → Delivered, live across both panels |
| 📱 Responsive | Mobile-first layout using Bootstrap grid |
| 🧩 Stack | `PHP` `MySQL` `JavaScript` `Bootstrap` `HTML/CSS` |

<br/>

### 📋 Student Attendance System
> *Institution-grade web-based attendance management with role-based access*

A structured attendance platform for educational institutions with multi-role environments — admins manage the system, teachers mark attendance per subject, students view their own records. Attendance percentages auto-calculated and flagged below minimum threshold. Records exportable for reporting and compliance.

| Feature | Details |
|:---|:---|
| 🎭 Role System | Separate dashboards for Admin, Teacher, and Student |
| ✅ Attendance Marking | Subject-wise daily attendance with date-range filtering |
| 📈 Auto % Calc | Real-time attendance percentage per student per subject |
| 📤 Export | Exportable records for administration and compliance |
| 🧩 Stack | `PHP` `MySQL` `Bootstrap` `JavaScript` |
